# Lab-Consumables-Equipment-Management-System
面向科研实验室打造的耗材及设备物资全生命周期管理系统，覆盖用户权限、组织实验室、物资主数据、仓储库位、采购入库、库存批次、借用归还、设备预约 / 维护、盘点、审批通知、审计合规、统计可视化 13 大核心模块。基于 23 张数据表实现主数据与事务数据分离、全流程事件流水追溯，支持精细化权限管控、多业务通用审批流、危化品合规管理、设备检定保养联动，同时提供库存周转率、设备利用率等多维统计分析与可视化，满足 GLP/GMP 实验室规范，实现物资从采购到报废的全链路可追溯与智能化管控。
# Lab-Consumables-Equipment-Management-System
**Laboratory Consumables & Equipment Management System | C# WinForm Project**

A desktop-based laboratory management system developed with **C#** and **WinForm**, providing full-process management for equipment, consumables, inventory, borrowing, approval, and data statistics.

The system adopts a modular architecture and database-driven design to realize paperless, intelligent, and standardized management of laboratory resources, suitable for schools, training rooms, and research labs.

## Project Overview
- Development: C# + WinForm
- Database: MySQL
- Architecture: Modular, desktop standalone system
- Scene: Laboratory material & equipment lifecycle management

## Core Functions
- User login & role permission management
- Equipment registration, borrowing, return, and maintenance
- Consumables application, approval, and distribution
- Inventory management, stock alert & inventory check
- Data query, statistics, and export
- System log & operation record traceability

## Technical Stack
- Language: C#
- Framework: .NET Framework / WinForm
- UI: Windows desktop interface
- Database: Data storage & table-driven management
- Tools: Visual Studio, ADO.NET data operation

## System Features
- Simple and stable desktop interface
- Clear authority management for different roles
- Complete process from purchase to scrap
- Real-time inventory and equipment status monitoring
- Data visualization and report export
- High security and easy to maintain
